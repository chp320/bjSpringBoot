17.1 MyBatis에서 파라미터 사용하기
- SQL문에 동적으로 변경되는 부분의 데이터를 파라미터로 어떻게 전달할까?
방법1) param 변수 뒤의 숫자를 늘려 가면서 사용
      ex. #{param1}, #{param2}, ...
방법2) 0부터 시작하는 인덱스 사용. (작성 시 편하지만 유지보수 시 값 의미 파악 어려움)
      ex. #{1}, #{2}, ...
방법3) @Param 어노테이션 사용해서 이름을 지정. (작성 시 번거롭지만 유지보수 시 명시적으로 의미 파악 가능)
      ex. public void 함수명(@Param("파라미터명") String 파라미터명, ...)
          [mapper 파일] select * from 테이블명 where 필드명 = #{파라미터명}
방법4) 해시맵을 사용하여 받는 방법    <-- 18장에서 다룰꺼임
